Селектори в CSS — це елементи, які використовуються для вибору HTML-елементів, до яких потрібно застосувати стилі.

Основні типи селекторів:
  1. Селектори тегів: Вибирають всі елементи конкретного типу. Наприклад, p вибирає всі абзаци.
  2. Селектори класів: Вибирають елементи за класом. Вони починаються з крапки (.). Наприклад, .apple вибирає всі елементи з класом apple.
  3. Селектори ідентифікаторів: Вибирають один елемент за унікальним ідентифікатором, що починається з решітки (#). Наприклад, #phone вибирає елемент з id phone.
  4. Поєднують декілька селекторів для точнішого вибору. Наприклад, div.box вибирає всі елементи div з класом box.
  5. Селектори атрибутів: Вибирають елементи за значенням атрибутів. Наприклад, input[type="email"] вибирає всі елементи input з атрибутом type, що має значення email.
